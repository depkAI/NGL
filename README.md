# NGL Clone

An anonymous messaging app clone built with Next.js.

## Features

- Create a link for anonymous messages
- Send anonymous messages
- Admin page to view all messages

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

- Go to the home page, enter a username, and create your link.
- Share the link to receive anonymous messages.
- Visit /admin to view all messages.
